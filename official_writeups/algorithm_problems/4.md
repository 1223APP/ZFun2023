## 可能的解法

### DP

对于背包问题的介绍可以参考 [这篇](https://oi-wiki.org/dp/knapsack/)。

物品 = 技能；技能只能使用 1 次 = 0/1 背包；技能可以使用多次 = 完全背包。

使用 Python 的解法可以参考 [这个](https://github.com/NoSparkHere/ZFun2023/blob/main/problems/algorithm_problems/std/4.py) 示例程序。
